# RSA Encryption 

This project demonstrates the use of RSA keys to encrypt a message using Python. The `rsa` library is used to handle the RSA encryption and decryption process.

## Prerequisites

- Python 3.x
- `rsa` library

You can install the `rsa` library using pip:

```bash
pip install rsa
